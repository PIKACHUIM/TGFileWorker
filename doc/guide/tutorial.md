# 功能教程

## 1. 添加来源

进入后台 → **来源管理** → **添加来源**：

| 字段 | 说明 | 是否必填 |
|------|------|---------|
| 名称 | 来源显示名称，如"4K 电影频道" | 必填 |
| 频道 ID | TG 频道 ID（如 `-1001234567890`）或 `@用户名` | 必填 |
| 内容类型 | 视频 / 音频 / 图片 / 电子书 / 文件 | 必填 |
| MTProto API ID | 在 https://my.telegram.org/apps 申请 | 必填（扫描用）|
| MTProto API Hash | 同上 | 必填（扫描用）|
| Session String | 通过工具导出的 session 字符串 | 必填（扫描用）|
| Bot Token | BotFather 创建的 Token | 可选（直链用）|

::: warning
Bot Token 仅支持 ≤ 20MB 的文件生成直链，大文件请使用代理下载。
:::

## 2. 扫描频道

1. 在来源列表找到目标来源，点击 **扫描** 按钮
2. SSE 连接建立后实时显示进度：当前文件名、已处理数量
3. 扫描为增量模式：已扫描的消息不会重复处理
4. 如需重新全量扫描，先点 **清空来源媒体**，再点扫描

## 3. 媒体刮削

**单条刮削**：在媒体管理列表，点击对应条目的 **刮削** 按钮

**批量刮削**：点击顶部 **批量刮削** 按钮，只处理 `scraped_at IS NULL` 的条目

**刮削规则**：
- 视频 → 先 TMDB，失败则豆瓣
- 音频 → MusicBrainz
- 电子书 → 先 Google Books，失败则豆瓣读书
- 图片/文件 → 不自动刮削，仅支持手动编辑

**配置刮削数据源**：后台 → **系统设置** 中填入 TMDB API Key 和豆瓣 Cookie。

## 4. 手动编辑媒体信息

在媒体管理列表点击 **编辑**，可修改：
- 标题
- 封面图片 URL
- 简介
- 发行日期
- 评分（0-10）
- 分类标签

## 5. 前台使用

### 浏览媒体
- 首页展示所有媒体，可按频道、类型筛选，支持关键词搜索
- 点击卡片进入详情页

### 在线播放
- 视频/音频：详情页点击 **在线播放**
- 视频使用 ArtPlayer，支持 HLS 流、画中画、全屏

### 下载文件
| 方式 | 说明 | 适用场景 |
|------|------|---------|
| 代理下载 | 通过 Worker 中转，支持 Range | 所有文件 |
| 直链下载 | 302 跳转 TG CDN | ≤ 20MB，需配置 Bot Token |
| STRM 文件 | 下载 `.strm` 文件 | Infuse / Emby / Jellyfin |
| 复制链接 | 复制代理流地址 | 外部播放器 |

### 使用 STRM 文件
1. 下载 `.strm` 文件
2. 放入 Infuse/Jellyfin 的媒体库目录
3. 播放器会直接请求 Worker 代理流

## 6. 系统设置

| 设置项 | 说明 |
|--------|------|
| TMDB API Key | https://www.themoviedb.org/settings/api 免费申请 |
| 豆瓣 Cookie | 登录豆瓣后从浏览器 F12 → Network → Cookie 中复制 |
| Worker 公开 URL | STRM 文件内容中使用的基础地址 |
